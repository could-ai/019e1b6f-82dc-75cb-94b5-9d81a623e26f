# The Urban Foundry - Premium Restaurant Website

A luxury, modern, high-end restaurant and nightlife website designed for "The Urban Foundry" located in Baner, Pune.

## Product Overview
This application serves as the digital front door for a premium industrial-themed bar and dining space. It features a cinematic and elegant design with a dark luxury theme, smooth scroll effects, and responsive sections highlighting the menu, nightlife experience, and table reservations.

## Main User Flows
- **Landing (Hero):** Immersive visual of the restaurant's ambience, with quick actions to view the menu or reserve a table.
- **About:** The narrative of The Urban Foundry, introducing its brand and industrial-themed aesthetic.
- **Menu Preview:** High-quality imagery highlighting signature cocktails, global tapas, and fine dining options.
- **Experience:** Insight into DJ nights, live music, and the nightlife energy.
- **Reservations:** Forms for booking tables and quick access to WhatsApp-based concierge.

## Tech Stack
- **Framework:** Flutter (Mobile, Desktop & Web)
- **Design:** Dark luxury theme leveraging Google Fonts (Playfair Display for headings, Lato for body).

## Setup & Run
1. `flutter pub get`
2. `flutter run -d chrome`
